Resume PDF Placeholder
=====================

This folder should contain a resume.pdf file that will be downloadable from the website.

The website has links to /resume.pdf in the following sections:
- Hero section: "Download Resume" button
- Experience section: "Download Full Resume" link
- Footer: "Resume" link

To add your resume:
1. Create or export your resume as a PDF file
2. Name it "resume.pdf"
3. Place it in this public folder
4. The links will automatically work

Current resume links point to: /resume.pdf
